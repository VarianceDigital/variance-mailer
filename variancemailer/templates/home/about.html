About this site

This site is the presentation part of the Variance Mailer.

It is an API service implementing a minimalistic mechanism to send emails to users who signed-up with the CatLoader application.

It uses MailerToGo Heroku add-on. In order to use the code, one needs a valid MailerToGo subscription, along with MailerToGo authentication parameters.
